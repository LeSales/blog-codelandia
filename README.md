# Blog-codelandia
Blog baseado no desafio da comunidade codelândia

## O que foi utilizado?
Para o desenvolvimento até o momento atual foi utilizado HTML5, CSS3(SASS) e JavaScript.

Todo layout foi feito com Flexbox.
